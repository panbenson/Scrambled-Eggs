# Scrambled Eggs
<b>Please Note!</b>
This game is written in Python 2.7 and uses the Pygame module, which can be downloaded <a href = "http://www.pygame.org/download.shtml">here.</a>

Scrambled Eggs is a remake of the popular mobile game Flappy Bird.
